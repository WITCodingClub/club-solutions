# WIT Coding Club Solutions
This repository holds the solutions for the various challenges completed during WITCC meetings, and from various members of the club.

## How to submit

1. Create a new branch [here](https://github.com/WITCodingClub/club-solutions/branches)
    - Name it the following: ``YOURNAME-YYYY-MM-DD``
2. Open your new branch in GitHub Desktop (click the green "Code" button, then click "Open with GitHub Desktop")
3. Go to the folder of the meeting day (click "Show in Explorer" in GitHub Desktop for easy access) and add your solution
    - Create a new folder with your name and add your solution there
4. Push your changes
5. Create a Pull Request (there will be a blue button in GH Desktop)

## How to check your solution

Go to the folder of the meeting day, then find the folder that has "- Main" at the end. This folder contains the master input and solution which you can use to check your work. If you have questions or need assistance, DM Logan or Daniel on Discord.
