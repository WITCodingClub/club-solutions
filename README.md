# WIT Coding Club Solutions
This repository holds the solutions for the various challenges completed during WITCC meetings, and from various members of the club.

## How to submit

1. Create a new branch at [here](https://github.com/WITCodingClub/club-solutions/branches)
    - Name it the following: ``YOURNAME-YYYY-MM-DD``
2. Open your new branch in Github Desktop (you may have to hit pull)
3. Go to the folder of the current day, and add your solution
    - NOTE: if your solution requires multiple files (generally java), then put it in a folder
4. Push your changes
5. Create a Pull Request (there will be a blue button in GH Desktop)
