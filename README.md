# WIT Coding Club Solutions
This repository holds the solutions for the various challenges completed during WITCC meetings, and from various members of the club.

## How to submit

1. Clone this repo in GitHub Desktop (click the green "Code" button, then click "Open with GitHub Desktop")
2. Create a new branch [here](https://github.com/WITCodingClub/club-solutions/branches)
    - Name it the following: ``YOURNAME-YYYY-MM-DD``
3. Open your new branch in GitHub Desktop (you may have to hit pull)
4. Go to the folder of the current day
5. Create a new subfolder, title it ``YOURNAME``, and add your solution there
6. Push your changes
7. Create a Pull Request (there will be a blue button in GitHub Desktop)
